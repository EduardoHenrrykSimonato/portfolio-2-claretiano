# Controle de Chamados Técnicos

## Objetivo
Desenvolver um aplicativo mobile em Ionic/Angular para controle de chamados técnicos, como parte da entrega acadêmica do Portfólio 2.

## Tecnologias utilizadas
- Ionic
- Angular
- TypeScript
- HTML
- SCSS
- Arrays (Armazenamento em memória)
- Services Angular
- Antigravity

## Instalação e Execução

Para instalar as dependências, navegue até a pasta do aplicativo e execute:
```bash
cd app-controle-chamados-tecnicos
npm install
```

Para executar o projeto, utilize:
```bash
ionic serve
```
*(Ou alternativamente: `npx ionic serve`)*

> **Observação Importante:** Este projeto **não utiliza banco de dados**. Todos os dados ficam armazenados apenas em arrays em memória durante a execução do aplicativo. Recarregar a página resetará os dados.
